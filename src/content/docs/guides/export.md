openNotes offers multiple ways to export your meeting data for sharing, archiving, or integration with other tools.

## Export formats

### PDF

Export a complete meeting document as a formatted PDF file. The PDF includes:

- Meeting title and date
- Full summary (overview, key points, decisions, action items)
- Complete transcript

**How to export as PDF:**
1. Open a meeting from the library
2. Click the **Export** button
3. Select **PDF**
4. Choose a save location

### Markdown

Export the meeting as a Markdown file (`.md`), perfect for:

- Pasting into Notion, Obsidian, or other note-taking apps
- Including in project documentation
- Version-controlling meeting notes in Git

**How to export as Markdown:**
1. Open a meeting from the library
2. Click the **Export** button
3. Select **Markdown**
4. Choose a save location

### Clipboard

Copy the meeting summary directly to your clipboard for quick pasting:

1. Open a meeting from the library
2. Click the **Copy** button
3. Paste into any app (email, Slack, docs, etc.)

The clipboard format is plain text with Markdown formatting.

## Bulk export

Export multiple meetings at once as a ZIP archive:

1. Go to the **Library** tab
2. Select the meetings you want to export
3. Click **Export Selected**
4. Choose between PDF or Markdown format
5. Save the ZIP file

The ZIP file contains one file per meeting, named with the date and meeting title.

## Export content

All export formats include:

| Section | Included |
|---------|----------|
| Meeting title | Yes |
| Date and time | Yes |
| Summary overview | Yes |
| Key points | Yes |
| Decisions made | Yes |
| Action items | Yes |
| Full transcript | Yes |

## Tips

- **Share summaries, not transcripts** — For quick meeting recaps, copy just the summary to clipboard
- **Use Markdown for integration** — Most project management and note-taking tools accept Markdown
- **Bulk export for backup** — Periodically export all meetings as a ZIP for offline archival
