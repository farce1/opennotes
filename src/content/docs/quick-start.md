This guide walks you through recording your first meeting with openNotes.

## 1. Install Ollama

openNotes uses Ollama to run AI models locally. Download and install it from [ollama.com](https://ollama.com).

## 2. Launch openNotes

Open the app. On first launch, it will guide you through pulling a recommended AI model (phi4-mini).

## 3. Start Recording

Click the Record button, select your microphone, and start your meeting. openNotes will transcribe in real-time.

## 4. View Your Summary

When you stop recording, openNotes generates a structured summary with key points, decisions, and action items.
