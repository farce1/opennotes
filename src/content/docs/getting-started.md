Welcome to **openNotes** — an open-source desktop app that records, transcribes, and summarizes your meetings entirely on your machine. No cloud services, no subscriptions, no bots joining your calls.

## What is openNotes?

openNotes is a local-first meeting notes application. It captures audio from your microphone or system audio, transcribes speech in real-time using an on-device AI model, and generates structured meeting summaries with key points, decisions, and action items.

Everything runs locally:
- **Transcription** uses [Parakeet TDT](https://huggingface.co/nvidia/parakeet-tdt-0.6b-v2), a state-of-the-art speech recognition model that runs directly on your CPU
- **Summarization** uses [Ollama](https://ollama.com) to run large language models locally — no API keys or internet connection required
- **Storage** uses SQLite — your meetings are saved as a single database file on your disk

## Key features

- **One-click recording** — Start recording with a single click, select your microphone, and go
- **Offline transcription** — Real-time speech-to-text powered by Parakeet TDT, no internet needed
- **AI-powered summaries** — Structured meeting notes generated by local LLMs via Ollama
- **Meeting library** — Browse, search, and manage all your recorded meetings
- **Flexible export** — Export to PDF, Markdown, or clipboard; bulk export as ZIP
- **Cross-platform** — Native app for macOS, Windows, and Linux

## Prerequisites

Before installing openNotes, make sure you have:

- A computer running **macOS 12+**, **Windows 10+**, or **Linux** (x86_64)
- **4 GB RAM** minimum (8 GB recommended for better LLM performance)
- **~3 GB disk space** for the app and default AI models
- A working **microphone** (built-in or external)
- [Ollama](https://ollama.com) installed for AI summarization (the app guides you through this on first launch)
