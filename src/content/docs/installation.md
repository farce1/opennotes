---
title: Installation
description: Install openNotes on macOS, Windows, or Linux
---

Download openNotes for your platform from the [GitHub Releases](https://github.com/nicodeforge/opennotes/releases) page.

## macOS

Download the `.dmg` file and drag openNotes to your Applications folder.

## Windows

Download the `.msi` installer and follow the installation wizard.

## Linux

Download the `.AppImage` file, make it executable, and run it.

## Next: Quick Start

Once installed, follow the [Quick Start](/quick-start/) guide to record your first meeting.
