## Tech Decisions

| Decision | Rationale |
|----------|-----------|
| Astro Starlight for docs | Purpose-built for docs, built-in search, static output, marketing page flexibility |
| Vercel for hosting | Free tier, auto-deploys from git, great Astro support |
| Zero-config Vercel deployment | Astro static output auto-detected, no vercel.json needed |
| Starlight root-mounted docs | `/docs/*` compatibility routes redirect to root-mounted pages |
| Landing page outside Starlight chrome | Full design control while reusing Starlight theme tokens |
| Lucide generic OS icons | Avoids brand icon availability issues in icon library |
