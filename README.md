# openNotes Documentation

Documentation and marketing site for [openNotes](https://github.com/nicodeforge/opennotes) — open-source AI meeting notes.

Built with [Astro](https://astro.build) and [Starlight](https://starlight.astro.build).

## Development

```sh
bun install
bun run dev
```

## Build

```sh
bun run build
```
